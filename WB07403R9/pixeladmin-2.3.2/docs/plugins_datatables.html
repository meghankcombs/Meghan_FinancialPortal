Usage

To use DataTables plugin, you need to include the next scripts:


<script src="path/to/js/libs/jquery.dataTables.js"></script>
<script src="path/to/js/libs/dataTables.bootstrap.js"></script>
<script src="path/to/js/pixeladmin/extensions/datatables.js"></script>
<script src="path/to/js/libs/angular-datatables.js"></script>

Then inject the plugin into your angular application:

angular.module('yourApp', ['datatables']);

Alternatively, you can include DataTables plugin using ocLazyLoad plugin:


$ocLazyLoad.load([
  {
    serie: true,
    name: 'datatables',
    files: [
      'path/to/js/libs/jquery.dataTables.js',
      'path/to/js/libs/dataTables.bootstrap.js',
      'path/to/js/pixeladmin/extensions/datatables.js',
      'path/to/js/libs/angular-datatables.js',
    ],
  },
]);
